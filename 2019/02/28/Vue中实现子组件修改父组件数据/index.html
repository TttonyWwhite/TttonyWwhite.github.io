子组件可以像使用自己data中数据一样使用props中的数据。但是要修改的话就会遇到问题了。查看文档可以知道，所有的prop都使得其父子prop之间形成了一个单向下行绑定，父级prop的更新会向下流动到子组件中，但是反过来则不行。这样会防止子组件意外改变父级组件的状态，从而导致你的应用的数据流向难以理解。
